Project Title 2: CODSOFT CALCULATOR PAGE DESIGN

The CodSoft Calculator project is a web-based tool that seamlessly combines 
functionality with an appealing interface. Developed using HTML, CSS, and JavaScript, the 
calculator provides users with an efficient solution for basic arithmetic operations. Its design 
features a pleasing linear gradient background, creating an aesthetically pleasing backdrop. 
The layout includes a clear input field for easy number entry and result visibility. Numeric and 
operational buttons are thoughtfully arranged, with the "C" button clearing the input and the 
"←" button removing the last digit.
User interaction is a key focus, allowing users to effortlessly input values by clicking on 
buttons. The "=" button triggers quick calculations using JavaScript's eval() function. This 
design prioritizes a straightforward and responsive tool for users' computational needs.
JavaScript functions power the calculator's functionality. The Solve(val) function 
dynamically appends clicked values to the input field, ensuring a responsive input mechanism. 
The Result() function evaluates expressions and presents results promptly. For user 
convenience, the Clear() function resets the input, while the Back() function intelligently 
removes the last entered digit. Together, these functions create a dynamic and user-friendly 
experience, establishing the CodSoft Calculator as a practical tool for on-the-fly calculations.
